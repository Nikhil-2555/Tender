# AI-Powered Tender Analysis

## Getting Started

1. Navigate to the project directory:
   ```bash
   cd tender-app
   ```

2. Install dependencies (if not already done):
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Features
- AI-Powered Tender Analysis
- Role-based Dashboards (Authority & Vendor)
- Secure Authentication
